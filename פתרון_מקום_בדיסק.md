```

```

## 驻转专转:

### 驻转专 1: 拽 拽 住拽

#### . 拽 拽爪 :
```powershell
# 拽 拽爪 
Remove-Item $env:TEMP\* -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item $env:LOCALAPPDATA\Temp\* -Recurse -Force -ErrorAction SilentlyContinue
```

#### . 拽 Docker ( 转拽):
```powershell
docker system prune -a -f
```

#### . 拽 pip cache:
```powershell
pip cache purge
```

#### . 拽 Python cache:
```powershell
Get-ChildItem -Path $env:LOCALAPPDATA\pip -Recurse -ErrorAction SilentlyContinue | Remove-Item -Force -Recurse -ErrorAction SilentlyContinue
```

## 拽转 拽 驻:

```powershell
# 拽 拽 住拽 C:
Get-PSDrive C | Format-Table Name, @{Label="Used(GB)";Expression={[math]::Round($_.Used/1GB,2)}}, @{Label="Free(GB)";Expression={[math]::Round($_.Free/1GB,2)}}

# 拽  转驻住 拽
Get-ChildItem C:\ -Directory | Sort-Object @{Expression={$_.GetFiles().Count}} -Descending | Select-Object -First 10 Name
```
